# Retail Store Sales — Cleaning, EDA & Tableau

## Overview
- This project explores retail transaction data using Python for data cleaning and EDA, and Tableau for visual analytics.

## Key Insights
- Top categories by total sales: Butchers, Electric household essentials, Beverages
- Online total sales are slightly higher than In-store; average spend is very similar
- Cash has the highest revenue and transaction count
- 2025 includes only January, so conclusions about 2025 trends are limited

## Files
- `retail_store_sales_data_cleaning.ipynb` (data cleaning + EDA)
- `retail_store_sales.csv` (main dataset)
- `retail_store_sales_cleaned.csv` (cleaned dataset)
